TimeBot Commands

Current commands as of 10/31/2025


These are some of the commands that I have implemented in the bot so far as of the current date posted above. I plan to add more commands in the future to enhance functionality and user experience.

Events & Calendar
Create, edit, and view upcoming sessions
/event create <title> <date/time> [location]
Core

Creates a new scrim, practice, or tournament event and syncs it to the connected Google Calendar.

Example: /event create "NU vs BU scrim 2025-10-12" 7:00 #val-scrims

/event list
Core

Shows all upcoming TimeBot events for this server, including date, time, and channel.

/event details <event_id>
Info

Displays full information for a single event, including description, participants, and any reminders.

/event delete <event_id>
Admin

Deletes an event from both Discord and the linked Google Calendar to keep schedules clean.

Player Availability
Find the best time for everyone
/availability set <day> <time start> <time end>
Players

Players submit when they are free on a given day (for example, practice blocks or scrim windows).

Example: /availability set Thursday 7:00 13:00

/availability view
Players

Shows your current availability on file so you can confirm what the coach sees.

/availability team
Coach

Aggregates player availability into a quick overview so coaches can see the best shared windows.

Reminders
Never miss another match
/reminder set <event_id> <minutes_before>
Core

Adds a timed reminder in Discord before an event starts, so players get pinged before queue time.

/reminder list
Players

Displays all active reminders you’ve set through TimeBot.

/reminder cancel <reminder_id>
Players

Cancels a reminder you no longer need if a match or practice gets rescheduled.

Settings & Misc
Per-player and per-server options
/profile link-calendar
Setup

Starts the process of linking your Google Calendar so TimeBot can sync events for you.

/timezone set <timezone>
Setup

Sets your personal timezone so event times display correctly, even for international teams.

/help
Info

Summarizes core TimeBot commands and links users back to this page for details.
